CHURCH UI

POSITION

UI/UX Designer

TECH USED IN THE PROJECT

HTML, CSS, Adobe XD

YEAR

2022

WORK

Design the web UI for the front-end developer

It is a static website.

The project is a static website created for a local church located in Laguna, Philippines. The website was built using HTML and CSS technologies and serves as a platform for the church to reach out to its members and the wider community.

About the color palette.

I decided to choose a red color palette as it is the primary color used by the church. This not only gives a personal touch to the website but also helps in creating a consistent branding for the church.

Some font colors are not that good on some areas of the website. With these, I have learned to carefully consider the color choices for text to ensure readability and accessibility for all users.
